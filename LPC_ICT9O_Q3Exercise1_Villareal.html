HTML

HyperText Markup Language

Hypertext Markup Language or HTML is the standard markup language for creating webpages. It has elements, which are called HTML elements, that enable the development and structuring of sections, paragraphs, and links. HTML also describes the structure or arrangement of a webpage.

CSS

Cascading Style Sheets

Cascading Style Sheets or CSS is used to define styles for our webpages' design, layout, and variants. It is used to control the look of a website. It is also used to format and organize webpages. Through the usage of external CSS, it permits the separation of style and content.

Bootstrap

Boostrap Version 4

Bootstrap is used to create a faster and easier web development. It gives us the ability to create webpages that are compatible with any device. May it be a phone, tablet, laptop, or desktop, the use of Bootstrap allows for easy use of webpages.
